# Claude Code Marketplace

这是一个 Claude Code Skills 集合项目，包含多个专业的前端开发和产品管理技能。

## 包含的 Skills

### 前端开发
- **react-component-generator** - React 组件生成器
- **feature-builder** - 完整功能构建器（UI + 业务逻辑 + API）
- **ui-analyzer** - UI 设计稿分析器
- **code-review** - 代码审查工具

### 辅助工具
- **request-analyzer** - 智能请求分析器
- **prompt-optimizer** - 提示词优化器
- **skill-creator** - Skill 创建工具

### 内容创作
- **tech-article-writer** - 技术文章写作助手
- **wechat-article-writer** - 微信公众号文章创作助手
- **product-manager** - 产品经理助手

## 技术栈

- React 18+ with TypeScript
- Next.js (App Router)
- Tailwind CSS
- Zustand (状态管理)
- React Query (数据获取)
- Zod (运行时验证)

## 使用方式

将整个 `.claude` 目录复制到你的项目根目录，Claude Code 会自动识别并加载这些 skills。
