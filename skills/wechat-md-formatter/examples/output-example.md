# Python 编程入门指南

这是一篇关于 Python 编程的文章，帮助你快速了解这门语言的特点和应用。

## 什么是 Python

Python 是一种高级编程语言，由 Guido van Rossum 在 1991 年发布。它的设计哲学强调代码的可读性和简洁的语法。

Python 支持多种编程范式，包括面向对象、命令式、函数式和过程式编程。

## 为什么选择 Python

Python 作为编程入门语言有以下优势：

- 语法简单易学
- 有丰富的第三方库
- 社区活跃，资源丰富
- 应用广泛

> 💡 提示：GitHub 上有超过 100 万个 Python 开源项目，你可以从中学习和借鉴。

## Python 的应用领域

Python 可以应用在多个领域：

1. Web 开发
2. 数据分析
3. 人工智能
4. 自动化脚本

### 代码示例

下面是一个简单的 Python 程序：

```python
def hello():
    print("Hello World")
    for i in range(10):
        print(i)

hello()
```

**代码说明**：这个程序定义了一个函数，打印 "Hello World" 并输出 0-9 的数字。
