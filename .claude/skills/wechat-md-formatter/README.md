# 微信公众号 Markdown 智能排版 Skill

## 简介

这是一个专门为微信公众号文章优化的 Markdown 排版工具。它可以将各种格式的文本内容（纯文本或 Markdown）智能转换为适合公众号阅读的格式。

## 主要功能

- ✅ 标题层级优化（避免层级跳跃，限制在 H1-H3）
- ✅ 段落间距优化（适合移动端阅读）
- ✅ 列表格式统一（统一符号，避免过度嵌套）
- ✅ 代码块美化（添加语言标识，提供截图建议）
- ✅ 表格优化（简化复杂表格，提供替代方案）
- ✅ 中英文混排优化（自动添加空格）
- ✅ 链接格式规范
- ✅ 公众号适配建议

## 使用方法

### 方法 1：直接调用

在 Claude Code 中输入：

```
/wechat-md-formatter

[粘贴你的文章内容]
```

### 方法 2：指定文件

```
/wechat-md-formatter 请优化这个文件：article.md
```

## 使用示例

查看 `examples/` 目录：
- `input-example.md` - 优化前的文章示例
- `output-example.md` - 优化后的效果

## 适用场景

- 技术文章转公众号发布
- 博客文章格式优化
- 纯文本内容添加格式
- 过度格式化内容简化

## 输出内容

使用这个 skill 后，你会得到：

1. **优化后的 Markdown 内容** - 完整的排版后文本
2. **排版优化说明** - 详细的调整说明
3. **公众号适配建议** - 针对性的发布建议
4. **检查清单** - 发布前的注意事项

## 注意事项

- 本工具优化的是 Markdown 格式，最终效果取决于公众号编辑器
- 复杂的代码块和表格建议转为图片
- 外部链接在公众号中有限制，建议使用"阅读原文"功能
